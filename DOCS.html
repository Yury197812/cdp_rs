📡 API Documentation

Базовый URL

http://localhost:8080

Эндпоинты

GET /health

Проверка здоровья сервера

curl http://localhost:8080/health
Status: 200 OK
GET /api/overview

Обзор системы: количество проектов, активных, завершённых

curl http://localhost:8080/api/overview
{"projects": 5, "active": 3, "completed": 2}
GET /api/projects/:id

Детали конкретного проекта

curl http://localhost:8080/api/projects/1
{"id": 1, "name": "Orchestrator LLM", "description": "...", "status": "active"}
GET /api/projects/:id/branches

Список веток проекта

curl http://localhost:8080/api/projects/1/branches
[{"id": 1, "name": "Analysis", "role": "worker"}, ...]
GET /api/projects/:id/tasks

Список задач проекта

curl http://localhost:8080/api/projects/1/tasks
[{"id": 1, "title": "Implement API", "status": "pending"}, ...]

Коды ответов

КодОписание
200Успех
400Неверный запрос
404Не найдено
500Внутренняя ошибка сервера
