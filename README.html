🚀 Orchestrator LLM

Система для массовой обработки задач в ChatGPT с управлением проектами и координацией веток

v4.9 Production Ready 145+ Tests

📦 Архитектура

orchestrator/
├── core/           # Ядро системы
│   ├── types.rs           # Типы данных
│   ├── project_manager.rs # Управление проектами
│   ├── branch_manager.rs  # Управление ветками
│   ├── chat_manager.rs    # Управление чатами
│   └── task_manager.rs    # Управление задачами
├── workers/        # Рабочие узлы
│   └── worker.rs          # Worker + WorkerPool
├── coordinator/    # Координация
│   ├── dependency_monitor.rs
│   ├── data_exchange.rs
│   └── sync_manager.rs
├── dashboard/      # Веб-интерфейс
│   ├── api.rs             # REST API
│   ├── ws.rs              # WebSocket
│   ├── stats.rs           # Статистика
│   └── server/mod.rs      # Axum сервер
└── database/       # Хранилище
    ├── sqlite/            # SQLite
    ├── pool/              # Пул соединений
    └── models/            # Модели данных

🚀 Быстрый старт

Docker
# Сборка и запуск
docker-compose up -d

# Проверка
curl http://localhost:8080/health
Локальная установка
# Клонирование
git clone https://github.com/Yury197812/cdp_rs.git
cd cdp_rs

# Сборка
cargo build --release

# Запуск
./target/release/orchestrator

📡 API Endpoints

МетодПутьОписание
GET/healthПроверка здоровья
GET/api/overviewОбзор системы
GET/api/projects/:idДетали проекта
GET/api/projects/:id/branchesВетки проекта
GET/api/projects/:id/tasksЗадачи проекта

📊 Примеры запросов

# Обзор системы
curl http://localhost:8080/api/overview

# Детали проекта
curl http://localhost:8080/api/projects/1

# Ветки проекта
curl http://localhost:8080/api/projects/1/branches

# Задачи проекта
curl http://localhost:8080/api/projects/1/tasks
